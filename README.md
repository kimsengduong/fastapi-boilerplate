# fastapi-boilerplate
FastAPI boilerplate

Clone:
```
git clone https://github.com/kimsengduong/fastapi-boilerplate.git
```

Virtual Environment
```
python -m venv venv
```

Active virtual environment
```
source venv/bin/active
```

Install dependencies
```
pip install -e .
pip install -r requirements.txt
```


run
```
fastrun
```
